    Predictive Model

    Python

    A retail company wants to understand their customer purchase behaviours against various products of different categories. They have shared purchased summary of various customers for selected high volume products from last month. We want to build a model to predict how much customers will spend on specific products on some future month, given their purchase history from some past month. This will help create personalized offers for customers against different products in the future. We used a distance measure of Root Mean Squared Error (RMSE) to score and compare our estimates on part of the training set that we have. We experimented with different regression models for best results with the dataset of over 550,000 instances.

    Project Leaders: Dr. Jessica Lin (CS professor at GMU), Aden Tran, and Bradley Dufour

    K-Mean Clustering

    Python

    Implemented the K-Means Clustering algorithm from scratch to create clusters from an input data consists of 10,000 images of handwritten digits (0-9). The images were scanned and scaled into 28x28 pixels. For every digit, each pixel can be represented as an integer in the range [0, 255] where 0 corresponds to the pixel being completely white, and 255 corresponds to the pixel being completely black. This gives us a 28x28 matrix of integers for each digit. We can then flatten each matrix into a 1x784 vector. No labels are provided. The task is to assign each of the instances in the input data to K clusters identified from 1 to K. Results were evaluated using V-Measure Metric.

    Project Leaders: Dr. Jessica Lin (CS professor at GMU)
